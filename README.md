# Panel Analyzer

A React app for analyzing panels.

## Setup

1. Copy `.env.example` to `.env` and fill in your API keys.
2. Install dependencies (if using GitHub Codespaces or any online editor):
   ```bash
   npm install
   ```
3. Start the app:
   ```bash
   npm start
   ```

## Features
- Upload files
- Display analysis results
- Easily configurable API endpoint
